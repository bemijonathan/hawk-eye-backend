## Hawk Eye Backend solution 

## Architecture 
- Model 
    - Users
    - Contact
    - Plan => social media
    - Alert
- Microservices
    - Notification Service (__event based__)
    - Image Upload Services
- Database
    - MongoDB
    - AWS S3 
    - Serverless
    - Heroku 
    - SNS

## Features

* Authenticate Users
* Find Users in the same Location 
* Upload Images
* Make Public Tweets
* Send Email and SMS
